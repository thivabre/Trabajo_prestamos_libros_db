# Trabajo_prestamos_libros_db
Aqui se presenta la realizacion de la base de datos referida a una biblioteca que realiza prestamos.
Este repositorio posee:
- La construccion y estructuracion de la base de datos
- Inserts de datos para su ejemplificacion
- Un procedure que devuelve los ids de los libros en prestamo y la fecha del prestamo
- El diagrama MER de la base de datos
- El diagrama fisico de la base de datos

# Branches
Existen 2 branches en las cuales subieron los archivos los distintos integrantes del equipo
- **Base de datos**: Aqui se subieron todos los archivos referidos con la base de datos
- **Documentacion**: aqui se subieron todos los archivos referidos con la documentacion de la base de datos
